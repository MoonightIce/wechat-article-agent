# Source Organization Prompt

Version: 0.1.0

Purpose: Turn raw materials into traceable source records.

Inputs:

- Source title or origin.
- URL when available.
- Excerpt or summary.
- Intended article/topic.

Output:

- Source ID.
- Source type.
- Verification status.
- Copyright risk.
- Supported claims.
- Usage notes.

Rules:

- Prefer summaries over long quotes.
- Flag platform/API/policy claims for primary-source verification.
- Mark private or copyrighted materials as needing human approval.
